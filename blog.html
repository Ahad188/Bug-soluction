1. Write the difference between map(),forEach(),filter(),or find() ?

Map() is an object that retrieves data one by one through an array and returns that data.
The forEach() is an object that retrieves data one by one through an array and does not return that data. It is the opposite of Map.
The function of the filter() is to give only the data that matches the condition
The function of find() is to return only the first data that matches the condition
